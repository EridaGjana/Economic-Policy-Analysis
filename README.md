
# Economic Policy Analysis (Time Series & Simulation)

## Overview
This project analyzes Eurostat macroeconomic time-series data to study the relationship between GDP and government spending. A Dynare model is included to simulate policy scenarios.

## Data
- Source: Eurostat
- Main variables: GDP, government expenditure

## Methods
- Time-series preprocessing and transformation
- Econometric analysis
- Policy simulation (Dynare)

## Tools
- GNU Octave / MATLAB (.m)
- Dynare (.mod)

## Repository Structure
- `data/` Eurostat dataset  
- `scripts/` analysis code  
- `model/` Dynare model  
- `report/` report/results
